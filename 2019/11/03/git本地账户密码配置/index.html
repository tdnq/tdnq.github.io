git本地账户密码配置

问题：全局配置本地git账户密码避免每次输入账号密码

依次执行下面命令

输入以下命令即可看见git的配置

git config --list --global

设置记住密码（输入一次密码后，以后的操作就不用再操作）

git config --global credential.helper store
