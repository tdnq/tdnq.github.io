包管理

npm发布包

npm发布条件

  • 注册npm账号且进行邮箱验证
  • 需要在npm源下操作，比如使用淘宝源时要切换回npm源

    步骤

  1. 在要发布的包下添加用户
    npm adduser
  2. 发布
    npm publish --access=public

    错误

  3. 无权限

以管理员权限运行cmd/powershell/终端

  1. unlink

删除C:/user/username下的”.npmrc”后重新执行步骤1

ps:最后发布成功后依旧有报错，不过邮件提醒成功
