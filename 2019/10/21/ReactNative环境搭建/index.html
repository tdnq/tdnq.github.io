ReactNative环境搭建

os: win10

官方文档

win10搭配RN环境需要node·8+,python·2+,jdk8

一，使用chocolatey安装需要软件

优点：方便可靠，且自动搭建环境变量; 不足：国内速度较慢，可使用科学上网工具或者代理

安装chocolatey(这里使用powershell安装)–官方文档

  1. 管理员权限打开powershell,执行下面命令
    Get-ExecutionPolicy
    如果返回“Restricted”,则执行下面命令
    Set-ExecutionPolicy AllSigned

  2. 执行下面命令

  3. 管理员权限重启powershell执行
    choco
    可看到chocolatey相关信息即安装成功

  4. 执行
    choco install -y nodejs.install python2 jdk8
    就会自动安装需要软件

  5. npm安装RN-cli
    npm install -g react-native-cli

安装Android Studio-下载连接

  1. 运行AndroidStudio安装程序,默认安装选项即可（安装位置推荐自定义）
  2. 默默等待安装后运行，进行自定义安装即可
