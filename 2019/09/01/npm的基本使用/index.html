包管理

npm的基本使用

常用命令

  • npm info package //返回包信息；
  • npm install package –global //全局安装；
  • npm install package –save //安装到依赖；
  • npm install package –save-dev //安装到开发依赖
  • npm outdated //检查更新；
  • npm update //更新；

其它：使用nrm切换源

  • npm install -g nrm //安装
  • nrm ls //列出可用源
  • nrm test //测试源
  • nrm use //切换源
